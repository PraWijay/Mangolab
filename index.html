Mango Lab

Automated Mango Grading & Quality Measuring System

Research Project

Scope

Research Problem

Farmers struggle to find markets and value for their crops, often giving up on plantations due to pests, plant diseases, and inability to overcome them. They also struggle to find markets for fertilizers and pesticides, lack knowledge of market prices, and struggle to determine the quality of their yields. This leads to economic losses and insufficient harvests

Research Gap

Existing systems for fruit quality and quantity assessment are suitable for offline environments but do not support farmers in marketing. The proposed project uses modern technologies to address these issues, improving farmers' mobility and earnings through online marketing. This project has been effective in supporting Sri Lankan farmers..

Research Objectives

The project aims to develop a machine learning-based approach to gather data from farmers, experts, and other resources, offering appropriate resources and data sharing. It includes creating a convenient dashboard for native farmers, developing pest identification and remedies, plant disease recognizers, market price analyzers, and fruit quality recognizers using image processing.

Methodology

he user-friendly mobile application has an intuitive UI that makes it simple for consumers to seek mango prices. The web service and the app may communicate with each other quickly and effectively thanks to the REST API that the app employs. The REST API is used to communicate with the web service when the end user submits a request through the mobile application. Request processing, the web service immediately responds to the user via the REST API. This gives the end user an immediate response to their request.
